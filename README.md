# Проекты машинного обучения на основе Decision tree

## Файлы


| №   | Ноутбук                        | Тема                                                                   |
| --- | ------------------------------ | ---------------------------------------------------------------------- |
| 1   | `1. Hotel reservations .ipynb` | Мини проект. Прогноз отмены бронирования отеля с помощью Decision Tree |
| 2   | `2. decision-tree-with-optuna.ipynb` | Подбор гиперпараметров `DecisionTreeRegressor` через Optuna на датасете бронирований отелей |


## Быстрый старт

1. Клонируйте проект.
2. Создайте виртуальное окружение и установите зависимости:
  ```bash
   python -m venv .venv
   source .venv/bin/activate   # или .venv\Scripts\activate в Windows
   pip install -r requirements.txt
  ```
3. Запускайте нужный `.ipynb` файл

## Стек

- phik, matplotlib, seaborn
